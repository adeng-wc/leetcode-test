# leetcode 算法练习

## 回溯算法

回溯算法的本质就是 _穷举_，只能通过 _剪枝_ 来优化。

回溯法解决的问题可以抽象成 _树形结构_。

因为回溯法解决的都是 _在集合中递归查找子集_，_集合的大小_ 就构成了 _树的宽度_，_递归的深度_ 就构成了 _树的深度_。

解决的问题：
1. _组合问题_：N个数里面按一定规则找出k个数的集合
2. _切割问题_：一个字符串按一定规则有几种切割方式
3. _子集问题_：一个N个数的集合里有多少符合条件的子集
4. _排列问题_：N个数按一定规则全排列，有几种排列方式
5. 棋盘问题：N皇后，解数独等

_组合_ 是 _不强调_ 元素顺序 的；_排列_ 是 _强调_ 元素顺序。

## 贪心算法

贪心的本质是选择每一阶段的 _局部最优_，从而达到 _全局最优_。

## 动态规划

适合解决 有很多重叠子问题。

动态规划中 _每一个状态_ 一定是由 _上一个状态_ 推导出来的，这一点区别于贪心。
